# Visor de Trabajos

Dashboard de ofertas laborales (Analista de Datos / BI, Santiago de Chile) encontradas por el buscador de trabajo con Claude Code.

**Ver online:** https://langab.github.io/visor_trabajos/viewer/

- `viewer/` — la página del visor
- `data/jobs.json` — ofertas exportadas desde el scraper

Se actualiza con `python3 tools/update_visor.py --push` desde el repo `ai-job-search`.
